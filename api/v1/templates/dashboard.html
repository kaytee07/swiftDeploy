ABOUT

My project was inspired by my curiosity to build something I hadn't tried before. I wanted to create an application that could run other applications and give results to users easily. So, I came up with the idea of a one-click server that uses Docker. It also has a simple web page where you can start applications whenever you need them. This project is all about making it simpler for people to run software, and that's what motivated me to work on it.
